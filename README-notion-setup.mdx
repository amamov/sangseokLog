# Notion 연결 설정

이 블로그는 Notion Database를 데이터 소스로 사용합니다.

## 1) 환경 변수 등록
- NOTION_TOKEN: Notion 내부 통합(Internal Integration) 토큰
- NOTION_DATABASE_ID: 데이터베이스 ID (기본값으로 제공된 링크를 사용 중)

> Next.js에서는 .env 파일을 지원하지 않습니다. Vercel 대시보드에서 Project Settings → Environment Variables에서 등록하세요. 등록 후 재배포가 필요합니다.

제공한 링크에서 추출한 Database ID를 기본값으로 사용합니다:

## 3) 권장 속성
다음 속성들이 있으면 더 정확한 매핑이 가능합니다. (없어도 동작하며 자동 추론/대체됩니다.)

- Title (title)
- Slug (rich_text) — 슬러그를 명시하면 URL이 고정됩니다.
- Published (checkbox) — 공개 여부
- Date (date) — 게시 날짜
- Tags (multi_select)
- Summary/Excerpt/Description (rich_text) — 목록 카드의 요약
- Cover — 페이지 Cover 또는 Files 속성 사용

## 4) 권한
Notion Integration이 해당 Database에 접근 권한이 있어야 합니다. Database 화면 오른쪽 상단 ••• → Add Connections 에서 통합을 연결하세요.
